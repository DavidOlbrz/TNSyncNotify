# TNSyncNotify

Benachrichtigt per Discord ueber den Status deiner TrueNAS Cloud Sync Tasks.

## Setup

1. Abhaengigkeiten installieren:

```bash
npm install
```

2. Umgebungsvariablen in `.env` setzen (siehe `.env.example`).

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

Der TypeScript-Build landet in `dist/`.

## Production Start

```bash
npm start
```
